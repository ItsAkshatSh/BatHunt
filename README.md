# Bat Hunter

**A cave-themed remake of Duck Hunt where you hunt bats instead of ducks!**

## About the Game

Welcome to **Bat Hunt**!

This is a remake of the classic **Duck Hunt**, but with a twist, 
instead of ducks, you're hunting **bats inside a cave**.

Test your reflexes and aim as bats fly through the darkness!

---

## Controls

| Action              | Input                  |
| ------------------- | ---------------------- |
| Move / Scroll World | **← / →** or **A / D** |
| Aim                 | **Mouse**              |
| Shoot               | **Left Click**         |
