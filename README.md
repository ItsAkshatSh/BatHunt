# Bat Hunter

Hey there!!

Welcome to my remake of **Duck Hunt**… but instead of ducks, you're hunting **bats in a cave**?!?!

## Controls

- **← / →** or **A / D** to scroll through the world  
- **Mouse** is Aim  
- **Left Click** is Shoot
